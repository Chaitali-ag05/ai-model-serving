## Contributing

Thanks for your interest in contributing to this project. A few quick guidelines:

- Bug reports: open an issue with steps to reproduce and expected behavior.
- Feature requests: open an issue describing the use case and desired API.
- Pull requests: fork the repo, create a feature branch, and open a PR against `main`.
- Keep changes focused and add tests where applicable.

Development

1. Backend

```bash
cd backend
python -m venv .venv
. .venv/bin/activate   # or .\.venv\Scripts\Activate.ps1 on Windows PowerShell
pip install -r requirements.txt
```

2. Frontend

```bash
cd frontend
npm install
npm run dev
```
